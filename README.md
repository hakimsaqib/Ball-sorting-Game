# Ball Sorting Game 🎮

A browser-based **Ball Sorting Puzzle Game** built with **HTML, CSS, and JavaScript**.

The objective is simple: sort colored balls into separate tubes so that each tube contains balls of only one color.

---

## 🎯 How to Play

1. Click a ball to select it
2. Click another matching colored ball
3. The selected ball moves to that tube
4. Keep sorting until every tube contains only one color
5. Win the game 🎉

## ⚙️ Game Logic

The game works by:

- Creating multiple rooms (tubes) dynamically
- Generating colored balls randomly
- Detecting ball selection
- Allowing moves only when colors match
- Checking all rooms for a completed sorted state
